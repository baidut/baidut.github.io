Basic Usage

Each input can either be:

  • image data e.g. I where I = imread('peppers.png')
  • name of an image file e.g. 'peppers.png'
  • Matlab objects that support imshow method e.g. an ImCtrl object
  • [] (an empty axes)
  • Matlab expression e.g. '(imhist(I))'
